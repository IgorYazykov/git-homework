Команды для использования git

  1. git init - инициализирует папку для работы с git
  2. git add - правильность написания: git add
  3. git commit - правильность написания: git commit -m "коментарий"
    фиксирует изменения и добавляет коментарий
  4. git status - позволяет увидеть изменения в проекте
  5. git reset - позволяет удалить изменения
    Данная команда имеет такие формы вызова:
    • soft - позволяет удалить сохранение
    • hard - позваляет удалить все сделаное до предведущего сохранения
  6. git push - правильность написания: git push -u origin master
    позволяет отправить сохраненные изменения в репозиторий
  7. git checkout название ветки - переключает на другую ветку
    • параметр : "-b название ветки" - позволяет создать ветку
  8. git branch - показывает все ветки
  9. git pull origin master - добавляет с второстепенной ветки все изменения в основную ветку (master)
